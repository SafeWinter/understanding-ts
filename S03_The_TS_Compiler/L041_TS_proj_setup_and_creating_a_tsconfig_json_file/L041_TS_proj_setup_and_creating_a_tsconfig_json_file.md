# L041 TypeScript Project Setup & Creating a tsconfig.json File

---



> [!tip]
>
> 使用版本：
>
> - `NodeJS`：`v20.16.0`
> - `TypeScript (tsc)`：`v5.5.4`
> - `BunJS`：`v1.1.2`（与 `NodeJS` 版本保持一致）



在项目根目录执行命令：

```bash
tsc --init
```

即生成一个 `tsconfig.json` 配置文件。

本章只重点介绍常用配置，各配置项详细用法参考官方文档：[TSConfig Reference](https://www.typescriptlang.org/tsconfig/)。
